# CMC_API_Parser_v2
CoinMarketCap API Parser for Raspberry Pi.
Outputs current market data such as price for cryptocurrencies such as Bitcoin, Ethereum, Ripple and Chainlink.

Supported Cryptocurrencies:
Bitcoin(BTC)
Ethereum(ETH)
Ripple(XRP)
Chainlink(LINK)
